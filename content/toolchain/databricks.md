``` brew tap databricks/tap

```


### [Set Up Databricks Apps development environment](https://docs.databricks.com/gcp/en/dev-tools/databricks-apps/configure-env)

``` pip3 install databricks-sdk
    pip3 install databricks-sql-connector
```
